很美（也很远）的地方

滴水湖 - 我要去露营！

度娘：滴水湖又名芦潮湖，位于浦东新区南汇新城镇，距离上海市中心约76公里，是南汇新城的中心湖泊，由德国GMP公司的总体规划设计。

临港新城的滴水湖因为在南汇嘴临海边上，所以有一片很广阔的天空，湖水也较清澈。你可以在滴水湖的露营区搭起帐篷，夜观星空，早迎日出。


南汇嘴观海公园

在滴水湖附近，上海的东南角，离市中心有80公里……

地址：上海市浦东新区南汇新城世纪塘路


Cloud Pavilion - 黄浦江畔的云展厅

看起来通透又轻盈的展厅，由丹麦建筑事务所Schmidt Hammer Lassen Architects设计的“云展厅”。

官网：The Cloud Pavilion

dezeen上的报道：Schmidt Hammer Lassen’s Cloud Pavilion is a glass-walled events space in Shanghai


好有趣啊！

1000 trees - 天安阳光半岛

位于苏州河畔，设计上海世博会英国馆的小哥 Heatherwick 在上海的另一个作品，概念诠释为“都市地貌学”，现在已经完工了，油管上还有航拍视频，觉得不如想象效果图那么美，有机会去看看！

地址：上海市普陀区莫干山路120号

Dezeen的报道：1000 trees
